= Oracle灾难恢复
:allow-uri-read: 


[role="lead"]
NetApp MetroCluster为任务关键型Oracle工作负载提供高度可用的零数据丢失解决方案。

此外、MetroCluster等集成解决方案还可以简化当今复杂的横向扩展Oracle数据库、应用程序和虚拟化基础架构。MetroCluster将多种外部数据保护产品和策略替换为一个简单的中央存储阵列。它可以在一个集群模式存储系统中提供集成的备份、恢复、灾难恢复和高可用性(HA)功能。
