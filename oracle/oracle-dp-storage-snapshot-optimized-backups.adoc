= Storage Snapshot优化的备份
:allow-uri-read: 


[role="lead"]
Oracle Snapshot Optimized Backups是一种时间点恢复方法，允许用户使用快照执行恢复，而无需将数据库置于热备份模式。

用户仍然可以在不使用备份模式的情况下从快照中恢复，但仅支持使用 RECOVERY AUTOMATIC 进行完全恢复。使用 SNAPSHOT TIME 方法指示快照创建时间的不完整恢复。
